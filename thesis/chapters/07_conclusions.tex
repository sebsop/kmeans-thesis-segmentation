\chapter{Conclusions and Future Work}\label{chap:conclusions}
{\itshape Real-time, GPU-accelerated video segmentation bridges the gap between high-level vision tasks and performance-constrained physical hardware. This concluding section summarizes the core software contributions of this work, reviews the empirical performance and quality findings of both clustering engines, and outlines key future research directions to improve temporal stability and explore physical quantum hardware integration.}

\section{Summary of Contributions}
This work developed and evaluated a real-time, GPU-accelerated 5D image and video segmentation pipeline integrating classical and quantum-inspired K-Means engines. The primary contributions of this thesis are:
\begin{enumerate}
    \item \textbf{High-Performance CUDA Pipeline:} Developed an optimized 5D preprocessor and solver in CUDA C++ operating on the BGR-XY manifold. By mapping threads directly to strided grids and caching centroids in low-latency shared memory, the implementation eliminates warp divergence and coalesces VRAM transactions, achieving a theoretical throughput of up to $750\text{ FPS}$ on an RTX 5070 Ti, well exceeding the $120\text{ FPS}$ real-time video target.
    \item \textbf{Quantum-Inspired Simulation:} Designed and simulated an angular phase-space similarity metric modeled on the multi-qubit state overlap of the quantum SWAP test. This backend projects normalized 5D features onto a high-dimensional Hilbert hypersphere via a $[0, \pi/4]$ bounded cosine-squared phase encoding.
    \item \textbf{Asynchronous Diagnostics:} Implemented an interactive Dear ImGui user interface dashboard and background benchmarking manager. This framework allows on-demand parameter modification (such as cluster count $K$ and stride $S$) and asynchronously calculates core execution metrics (latency, iteration counts, WCSS, Davies-Bouldin, and Monte Carlo-approximated Silhouette scores) on a background thread, preventing display loop stutter.
\end{enumerate}

\section{Key Empirical Findings}
Empirical evaluations show both engines are highly viable for real-time edge deployment, with core optimization latency under $3.9\text{ ms}$ ($259\text{ FPS}$) on an NVIDIA RTX 5070 Ti. The quantum-inspired backend achieves complete functional equivalence to classical Euclidean K-Means, producing identical boundaries and quality metrics under all lighting environments. This is guaranteed by the monotonic nature of the cosine-squared mapping over the $[0, \pi/4]$ phase range.

Moreover, compiling the angular distance function into the hardware intrinsic \texttt{\_\_cosf()} offloads execution to the GPU's SFUs. This maintains performance parity with standard ALUs, even enabling the quantum-inspired engine to run slightly faster under low-density loads.

\section{Future Research Directions}
Several research directions remain open:
\begin{itemize}
    \item \textbf{Temporal Tracking:} To resolve frame-by-frame initialization shifts (like the halo and detail loss in Frame $t+1$), future work will integrate Hungarian association or optical flow tracking to ensure inter-frame boundary consistency.
    \item \textbf{QPU Execution:} Porting the angular phase mapping to physical NISQ hardware would allow the SWAP test to run natively in $\mathcal{O}(1)$ complexity once state preparation latencies and QPU error rates subside.
    \item \textbf{Edge Profiling:} Benchmarking the kernels on embedded platforms like NVIDIA Jetson to measure power efficiency and thermal behavior under edge constraints.
\end{itemize}
